Basic Commands

banned List of Banned Members from the server
info Shows the info of a person - ?info @[name]
rules List of Server Rules
serverinfo Shows information of the server
ping To check the ping of the bot

Moderation Commands

kick Kick a user from the server
ban Ban a user from the server

Fun Commands

dice To roll a die
flip To flip a coin
hug To Hug someone - ?hug @[username or hug yourself ;(
nicememe Nice Meme!
rnsfw Random NSFW Picture - Use at your own risk!
nsfw NSFW - ?nsfw <blank> for the list of subreddits
roll To roll a number 1 to 10

Image Commands

gif Search for some giphy!
imgnew Shows the newest image from a specified subreddit
imgrand Shows the user a random image from imgur
imgsearch Allows the user to search for an image from imgur
imgtop Shows the top image from a specified subreddit

Music Commands

play Plays a song
pause Pauses the currently played song
resume  Resume the currently played music
skip To skip a song
stop To stop a song and leaves the voice channel
disconnect Stop playing audio and disconnect the voice channel
summon Summons the bot to join your voice channel
volume Sets the volume of the currently playing song
join Joins a voice channel
songinfo Shows the info of the currently playing song
